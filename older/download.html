Project requirements

To successfully run bzip2-cuda you need the following things:

  • gcc 4.3 and above
  • NVidia GPU 8 series and above
  • NVidia basic and developers drivers for your system
  • CUDA SDK 3.2
  • Thrust v1.3.0
